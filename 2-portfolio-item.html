Real-time Chat App

React.js Firebase(storage+auth)

Source Code: https://github.com/pco111/200805_react_messenger

Web App for chat: https://messenger-3a1c7.web.app

This is a full-stack web application for chat built with React.js and Google Firebase. While firebase provides storage and user authentification, the front end displays the information stored in the facebook-messenger fashion.

How it works? After the user logs in from the registration page, he can input a message. Upon receiving the message from the input field, the frontend stores this new message into the database collections. The React components listening to changes in the database then displays this new message.
