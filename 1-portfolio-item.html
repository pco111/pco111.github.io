MERN full-stack ToDo List

API Redux React MongoDB NodeJS Express Heroku Bootstrap JWTauth

Source Code: https://github.com/pco111/200820_todo_list/tree/master

Todo List Web App: https://mern200820shoppinglist.herokuapp.com

This is a complete full-stack javascript web application based on the MERN stack. With an API fully-fledged with CRUD functions in the backend and react-redux in the frontend, this apparently simple application demonstrates my readiness for the full-stack development setting at work.

How it works? Explained here with the example of "adding items". Upon a click of button on the webpage, the front-end function makes an API call to the backend using axios with information about the new item wrapped in the API call. The backend parses the API request and makes changes in the MongoDB database and answers back with the "success" message. When the front end receives such message, it calls for an redux action to modify the global state in Redux(adding the item). Finally, the React components at the front end will display the changes as they notice the changes in the Redux state. Remarkably, JWT Authentification is also incorporated into the web app. You need to login in order to modify the todo list.
